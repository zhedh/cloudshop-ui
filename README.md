# Cloudshop UI

基于 antd 封装的业务 UI 组件库

## 介绍

Cloudshop UI 是一个轻量的，专门用于品牌云店业务的 UI 组件库。

目前接入使用的应用有：品牌云店后台、完美门店后、爱门店后台、基础服务后台。

## 快速开始

### 安装

```bash
# 通过 npm 安装
npm i cloudshop-ui

# 通过 pnpm 安装
pnpm i cloudshop-ui
```

### 引入组件

```tsx
import { ListSearch } from 'cloudshop-ui'
```

### 使用组件

```tsx
import { ListSearch } from 'cloudshop-ui'

function App(){
  return <ListSearch></ListSearch>
}

export default APP
```
