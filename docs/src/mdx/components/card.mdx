# Card 卡片

<section>

## 介绍

卡片容器，可承载文字、列表、图片、段落，以及可以自定义操作项和尾部。

</section>

<section>

## 引入

```tsx
import { Card } from 'cloudshop-ui'
```

</section>

```tsx
import { Button } from 'antd'
import { Card } from 'cloudshop-ui'

function App() {
  return (
    <Card
      title="默认卡片"
      actions={<Button>Action</Button>}
      footer={
        <div style={{ textAlign: 'right' }}>
          <Button>取消</Button>
          &nbsp;&nbsp;
          <Button type="primary">确定</Button>
        </div>
      }
    >
      <p>这是卡片内容...</p>
      <p>这是卡片内容...</p>
      <p>这是卡片内容...</p>
    </Card>
  )
}

export default App
```

</section>

## API

<section>

### Card 属性说明

| 属性      | 说明                                         | 类型      | 默认值 |
| :-------- | :------------------------------------------- | :-------- | :----- |
| title     | 卡片标题                                     | string    | -      |
| className | 自定义类名                                   | string    | -      |
| actions   | 卡片操作组，位置在卡片顶部右边，不设置不展示 | ReactNode | -      |
| footer    | 卡片底部自定义配置，不设置不展示             | ReactNode | -      |

</section>
