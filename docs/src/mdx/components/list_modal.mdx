# ListModal 列表弹层

<section>

## 介绍

列表弹层用弹层展示简单列表。

</section>

<section>

## 引入

```tsx javascript
import { ListModal } from 'cloudshop-ui'
```

</section>

## 代码演示

<section>

{props.basic}

```tsx
import { Button } from 'antd'
import { ListModal } from 'cloudshop-ui'
import { useState } from 'react'

function App() {
  const [visible, setVisible] = useState(false)

  const dataSource = [
    {
      key: '1',
      name: '胡彦斌',
      age: 32,
      address: '西湖区湖底公园1号',
    },
    {
      key: '2',
      name: '吴彦祖',
      age: 42,
      address: '西湖区湖底公园1号',
    },
  ]

  const columns = [
    {
      title: '姓名',
      dataIndex: 'name',
      key: 'name',
    },
    {
      title: '年龄',
      dataIndex: 'age',
      key: 'age',
    },
    {
      title: '住址',
      dataIndex: 'address',
      key: 'address',
    },
  ]

  return (
    <>
      <Button type="primary" onClick={() => setVisible(true)}>
        打开列表弹层
      </Button>
      <ListModal
        open={visible}
        rowKey="name"
        columns={columns}
        dataSource={dataSource}
        onCancel={() => setVisible(false)}
      />
    </>
  )
}

export default App
```

</section>

## API

<section>

### ListModal 属性说明

| 属性       | 说明                                      | 类型                               | 默认值 |
| :--------- | :---------------------------------------- | :--------------------------------- | :----- |
| open       | 对话框是否可见                            | boolean                            | -      |
| rowKey     | 表格行 key 的取值，可以是字符串或一个函数 | string \| function(record): string | key    |
| columns    | 表格列的配置描述，同 antd                 | string \| ColumnsType[]            | -      |
| dataSource | 数据数组，同 antd                         | object[]                           | -      |
| size       | 表格大小                                  | large \| middle \| small           | large  |
| pagination | 表格分页，同 antd                         | false \| TablePaginationConfig     | false  |
| onCancel   | 点击遮罩层或右上角叉或取消按钮的回调      | (e) =>void                         | -      |

</section>
